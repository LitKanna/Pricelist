# Everfresh Price List Editor

A simple web app to edit and download your wholesale price list as an image.

## Features
- Edit prices directly in browser
- Add/remove products and categories
- Download as professional PNG image
- Works on mobile and desktop
- Auto-saves your changes locally

## Usage
Visit the live app: [Everfresh Price List Editor](https://your-username.github.io/everfresh-price-editor/)

Simply open, edit, and download!
