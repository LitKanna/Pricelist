## Deployment

- **Production**: `index.html` - The stable, working version deployed to GitHub Pages
- **Testing**: `index-test.html` - Test new features here before promoting to production
- **Backup**: `index-production.html` - Backup copy of last known good version

## Testing Workflow

1. Make changes to `index-test.html`
2. Test on:
   - Desktop browsers
   - iPhone 17 Pro/Max
   - iPhone 16 Pro/Max
3. Once verified, copy `index-test.html` to `index.html`
4. Commit and push to deploy
